#  Brick Breaker — Agent RL

## Description
Projet de jeu Brick Breaker développé avec Unity, intégrant un agent d’apprentissage par renforcement (Reinforcement Learning).

## Objectif
Créer un agent autonome capable d’apprendre à jouer au jeu en optimisant ses actions grâce à une fonction de récompense.

## Technologies
- Unity
- C#
- Python
- ML-Agents
- Reinforcement Learning

## Structure
- Jeu-RL/ : projet Unity principal

## Compétences développées
- Modélisation d’environnement RL
- Définition des états, actions, récompenses
- Entraînement d’un agent autonome
- Intégration Unity / IA
